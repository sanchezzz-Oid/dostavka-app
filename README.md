# DostavkaApp
## Описангие проекта
DostavkaApp - это система онлайн заказов еды.
Проект состоит из мобильнго приложения, веб сайта, и серверного API.

## Структура репозитория
- MobileApp - мобильное приложение
- ServerApi - серверная часть
- Web - веб-сайт
